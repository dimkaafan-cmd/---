Проект: Tic-Tac-Toe "Loot & Win" Edition
Движок: Defold (Lua)
Описание: Классическая игра 3х3 с мета-механикой. За каждую победу игрок получает случайную награду разной редкости (Common, Rare, Legendary) и золото.
Особенности:
Полный игровой цикл с возможностью перезапуска.
Система выпадения лута на основе вероятностей.
Адаптивный GUI для отображения прогресса.


Минимальная UML-диаграмма

[ Input ] --(touch)--> [ controller.script ]

                              |
       +----------------------+----------------------+
       | (play_animation)     | (show_win)           | (restart)
       v                      v                      v
 [ cell_0...8 ]         [ game_ui.gui ]        [ controller.script ]
 (Отображение X/O)      (Экран награды)        (Сброс board = {0...})


 Какие ИИ-агенты использовались

В данном воркфлоу использовался один мультимодальный ИИ-ассистент (модель уровня Claude 3.5 Sonnet / GPT-4o).
Роль: Senior Game Developer & Defold Expert.
Задача: Генерация логики на Lua, помощь в навигации по интерфейсу Defold и отладка визуальных компонентов (спрайтов/атласов).


Примеры промптов и описание Workflow

Контекст воркфлоу: Я использовал итеративный подход «от простого к сложному». Вместо того чтобы просить «сделай игру целиком», я разбивал задачи на атомарные части: архитектура -> ввод -> логика -> мета-гейм.
Примеры промптов:

	На архитектуру:
	"Как правильно организовать структуру папок в Defold для маленькой игры, чтобы не смешивать ассеты и логику? Пришли пример структуры."

	На логику (алгоритм):
	"Напиши функцию на Lua для проверки победителя в массиве из 9 элементов (0, 1, 2) для игры в крестики-нолики."
		Это чистая математическая задача, ИИ выдал оптимизированный перебор всех линий.

	На интеграцию (Message Passing):
		"Как передать данные из скрипта логики в GUI скрипт в Defold? Нужно отправить название награды и количество золота."
		Промпт специфичен для движка (msg.post), ИИ сфокусировался на синтаксисе адресации объектов.

	На отладку интерфейса:
		"В Defold я выбрал атлас в спрайте, но картинка не появилась. Какие 3 причины проверить в первую очередь?"
		Вместо общего ответа ИИ выдал чек-лист (Z-координата, Default Animation, Scale кодировка ).
